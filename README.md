\## üìä Insights so far



\- Missing data can be effectively handled using mean/mode imputation or dropping strategy depending on attribute importance.  

\- Feature scaling (standardization and normalization) improves comparability of attributes.  

\- Engine power and make (for cars) show strong correlation with price.  

\- In laptops, features like \*\*RAM\*\*, \*\*processor speed\*\*, and \*\*screen size\*\* significantly impact pricing.
